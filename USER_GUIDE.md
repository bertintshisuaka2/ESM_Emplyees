# Employee Management System - User Guide

## Overview

The Employee Management System is a comprehensive web application designed to help organizations manage employee information, track workplace accidents, and store employee documents securely. The application is fully responsive and works seamlessly on desktop, tablet, and mobile devices.

## Features

### 1. Employee Management
- **Add New Employees**: Store complete employee information including personal details, contact information, position, department, and employment status
- **Search & Filter**: Quickly find employees by name, email, department, or position
- **Edit Information**: Update employee details at any time
- **View Details**: Access comprehensive employee profiles
- **Status Tracking**: Monitor employee status (Active, Inactive, Terminated)

### 2. Accident Reporting
- **Report Accidents**: Document workplace accidents with detailed information
- **Severity Levels**: Classify accidents by severity (Minor, Moderate, Severe, Critical)
- **Witness Information**: Record witness statements and details
- **Treatment Records**: Document medical treatment provided
- **Employee Association**: Link accidents to specific employees

### 3. Document Management
- **Upload Documents**: Store employee-related documents securely in the cloud
- **Document Categories**: Organize documents by type (Contract, Certification, Medical, Identification, Other)
- **Download Access**: Retrieve documents anytime from anywhere
- **Employee Association**: Link documents to specific employees

## Getting Started

### Accessing the Application

1. Open your web browser (Chrome, Firefox, Safari, or Edge)
2. Navigate to the application URL provided by your administrator
3. Click "Sign in" to authenticate using your Manus account

### Navigation

The application uses a sidebar navigation menu with three main sections:

- **Employees**: Manage employee records
- **Accidents**: Track and report workplace accidents
- **Documents**: Upload and manage employee documents

On mobile devices, tap the menu icon (☰) to access the navigation menu.

## Using the Application

### Managing Employees

#### Adding a New Employee

1. Click the "Add Employee" button in the top-right corner
2. Fill in the required fields:
   - **First Name** (required)
   - **Last Name** (required)
3. Fill in optional fields as needed:
   - Email
   - Phone
   - Position
   - Department
   - Hire Date
   - Salary
   - Address
   - Emergency Contact
   - Status (Active/Inactive/Terminated)
4. Click "Add Employee" to save

#### Editing an Employee

1. Find the employee in the list
2. Click the edit icon (pencil) in the Actions column
3. Update the information as needed
4. Click "Update Employee" to save changes

#### Searching for Employees

1. Use the search box at the top of the employee list
2. Type any part of the employee's name, email, department, or position
3. The list will automatically filter as you type

#### Deleting an Employee

1. Find the employee in the list
2. Click the delete icon (trash) in the Actions column
3. Confirm the deletion when prompted

**Note**: This action cannot be undone.

### Reporting Accidents

#### Creating an Accident Report

1. Navigate to the "Accidents" section
2. Click the "Report Accident" button
3. Fill in the required fields:
   - **Employee**: Select the affected employee
   - **Accident Date**: Date and time of the incident
   - **Severity**: Choose Minor, Moderate, Severe, or Critical
   - **Description**: Detailed description of what happened
4. Fill in optional fields:
   - Location
   - Witnesses
   - Treatment Provided
5. Click "Submit Report" to save

#### Viewing Accident Reports

All accident reports are displayed in a table showing:
- Employee name
- Date of accident
- Location
- Severity level (color-coded)

#### Editing an Accident Report

1. Find the report in the list
2. Click the edit icon (pencil)
3. Update the information
4. Click "Update Report" to save

#### Understanding Severity Levels

- **Minor**: Small injuries requiring minimal treatment (cuts, bruises)
- **Moderate**: Injuries requiring medical attention (sprains, minor burns)
- **Severe**: Serious injuries requiring immediate medical care (fractures, deep cuts)
- **Critical**: Life-threatening injuries requiring emergency response

### Managing Documents

#### Uploading a Document

1. Navigate to the "Documents" section
2. Click the "Upload Document" button
3. Select the employee the document belongs to
4. Choose the document category:
   - Contract
   - Certification
   - Medical
   - Identification
   - Other
5. Click "Choose File" and select the document from your device
6. Click "Upload" to save

**Supported File Types**: PDF, Word documents, images (JPG, PNG), and other common file formats

#### Downloading a Document

1. Find the document in the list
2. Click the download icon (arrow down)
3. The document will open in a new tab or download to your device

#### Deleting a Document

1. Find the document in the list
2. Click the delete icon (trash)
3. Confirm the deletion when prompted

**Note**: This action cannot be undone.

## Mobile Usage

The application is fully optimized for mobile devices:

### Mobile Navigation
- Tap the menu icon (☰) in the top-left to open the navigation menu
- Tap anywhere outside the menu to close it
- The current page is highlighted in the menu

### Mobile Features
- All tables are responsive and adapt to smaller screens
- Forms are optimized for touch input
- Search and filter functions work the same as on desktop
- File uploads work directly from your mobile device's camera or file storage

### Tips for Mobile Users
- Use landscape mode for better table viewing
- Swipe to scroll through long lists
- Tap and hold on table rows for additional options (browser dependent)
- Use the search function to quickly find specific records

## User Roles

The system supports multiple user roles:

### Admin Users
- Full access to all features
- Can add, edit, and delete employees
- Can create and manage accident reports
- Can upload and delete documents

### Regular Users
- Can view employee information
- Can create accident reports
- Can upload documents
- Limited deletion capabilities (based on permissions)

## Best Practices

### Employee Records
- Keep employee information up to date
- Update status when employees leave or change positions
- Include emergency contact information for all employees
- Verify email addresses and phone numbers for accuracy

### Accident Reporting
- Report accidents as soon as possible after they occur
- Be detailed and specific in descriptions
- Include all witness information
- Document all treatment provided, even minor first aid

### Document Management
- Use clear, descriptive file names
- Choose the correct category for each document
- Upload documents promptly after receiving them
- Regularly review and update expired certifications

## Security & Privacy

### Data Protection
- All data is stored securely in encrypted databases
- Documents are stored in secure cloud storage
- Access is controlled through authentication
- User actions are logged for audit purposes

### Password Security
- Use strong, unique passwords for your account
- Never share your login credentials
- Log out when using shared devices
- Report any suspicious activity to your administrator

## Troubleshooting

### Common Issues

**Cannot log in**
- Verify you're using the correct login credentials
- Clear your browser cache and cookies
- Try a different browser
- Contact your administrator if the issue persists

**Page not loading**
- Check your internet connection
- Refresh the page (F5 or Cmd+R)
- Clear browser cache
- Try a different browser

**Upload failing**
- Check file size (maximum 16MB for most files)
- Verify file format is supported
- Check your internet connection
- Try uploading a smaller file

**Search not working**
- Clear the search box and try again
- Check spelling
- Try searching with fewer characters
- Refresh the page
